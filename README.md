#BBP-Frontend

## Getting Started

Run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Using Docker

Build your container:

```bash
docker build -t nextjs-docker .
```

Run your container:

```bash
docker run -p 3000:3000 nextjs-docker
```
